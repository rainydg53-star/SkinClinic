💡 프로젝트 소개

누리클리닉은 사용자가 자신의 피부 상태에 적합한 시술 정보를 쉽고 편리하게 제공받을 수 있도록 하고,

상담부터 예약·결제까지 하나의 서비스에서 이용할 수 있도록 할수있는

3D 얼굴 진단 기술을 활용해 피부 상태를 분석하고, 그 결과를 바탕으로 

맞춤형 시술 추천, 상담, 예약 및 결제 기능을 제공하는 스킨클리닉 웹 플랫폼입니다.
